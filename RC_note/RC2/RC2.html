Basic Structure

Induction is a rather powerful tool in proof. The core philosophy is that some things are not constructed randomly but elaborately-designed. Therefore, we can find inner pattern. Everything starts from the simplest situation, which is base case.

(1) base case: P(0) is correct(1) \ \text{base case: P(0) is correct}(1) base case: P(0) is correct

Then, things get more complicated (for natural number, number is getting bigger; string may get longer), so we have inductive case.

(2) inductive case: ∀ n∈N,(P(n)→P(n+1))(2) \ \text{inductive case: } \forall \ n \in \mathbb{N}, (\text{P(n)} \to \text{P(n+1)})(2) inductive case: ∀ n∈N,(P(n)→P(n+1))

What's important that in inductive case, P(n)\text{P(n)}P(n) is the Inductive Hypothesis.

Therefore, the basic structure of induction is

  1. Base Case,
  2. Inudctive Hypothesis,
  3. Inductive Case

  • Structural Induction
    1. From my point of view, I would like to say both induction follow the same philosophy (the construction of something like string and natural number follows a pattern), but they use different characteristic of that pattern.

Equinumerosity

Definition: A set AAA is equinumerous to a set BBB (written A≈BA \approx BA≈B) if there is a bijection from AAA to BBB.

Theorem: For any sets AAA, BBB, and CCC:

  • ⊢A≈A\vdash A \approx A⊢A≈A.
  • A≈B⊢B≈AA \approx B \vdash B \approx AA≈B⊢B≈A.
  • (A≈B∧B≈C)⊢A≈C(A \approx B \land B \approx C) \vdash A \approx C(A≈B∧B≈C)⊢A≈C.

While you may not directly face problems proving that one set is equinumerous to another, it's still important to review some proofs in your homework and slides.

Some example you need to remember:

  • N≈Z\mathbb{N} \approx \mathbb{Z}N≈Z
  • N×N≈N\mathbb{N} \times \mathbb{N} \approx \mathbb{N}N×N≈N
  • Q≈N\mathbb{Q} \approx \mathbb{N}Q≈N

These examples all demonstrate a graphical way of thinking. You should keep that in mind.

Dominate

Definition : A set AAA is dominated by a set BBB (written A⪯BA \preceq BA⪯B) if there is an injection from AAA to BBB.

Countable Sets

Definition: A set AAA is countable if A⪯NA \preceq \mathbb{N}A⪯N. Otherwise, it is called uncountable.

Cantor-Schröder-Bernstein Theorem

For sets AAA, BBB, and CCC,

  • Reflexivity: ⊢\vdash⊢ AAA ⪯\preceq⪯ AAA.
